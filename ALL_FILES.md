# All Research and Skill Files

This repository contains the following essential files for the Wellness Companion skill:

- `README.md` – Overview and description of the skill.
- `install.md` – Installation guide.
- `CRISIS_PROTOCOL.md` – Full crisis detection, escalation, and response protocol (15 sections).
- `SKILL_PLAN.md` – Architecture, roadmap, and implementation plan.
- `wellness-config.json.template` – User configuration template.
- `prompts/`
  - `crisis_templates.txt` – Crisis message templates.
  - `daily_checkin.txt` – Daily check‑in prompts.
  - `weekly_summary.txt` – Weekly summary template.
- `tools/`
  - `coping_library.json` – CBT/ACT/DBT techniques library.

These files constitute the complete research and skill implementation needed for the wellness‑skill.
